API для интеграции

REST API для получения данных каталога продукции ТРЕК. Формат ответа — JSON.

Список всех товаров
GET /api/products
Товар по артикулу
GET /api/products/TR-2108
Пример ответа
{
  "sku": "TR-2108",
  "name": "Шаровая опора ТРЕК",
  "category": "ball_joint",
  "price": 890,
  "inStock": true,
  "stockQty": 1250,
  "description": "Шаровая опора для автомобилей ВАЗ...",
  "specs": [
    { "name": "Применяемость", "value": "ВАЗ 2108-2115" }
  ]
}

Ответ содержит массив объектов products и массив categories. Каждый товар включает артикул (sku), название, категорию, цену, наличие, остаток, описание и массив характеристик (specs).
