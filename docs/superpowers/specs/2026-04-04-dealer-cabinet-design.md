# Кабинет дилера ТРЕК — Спецификация

## Обзор
Веб-приложение "Кабинет дилера" для компании ТРЕК (производитель автозапчастей подвески). Демо/MVP для показа. Контрагенты (дилеры и СТО-партнёры) видят информацию о заказах, подают рекламации, работают с документами, участвуют в геймификации.

## Аудитория
- Крупные дилеры/дистрибьюторы (оптовые закупки, сложная аналитика)
- Мелкие СТО-партнёры (базовые функции, простота)
- Общий кабинет с разным уровнем доступа в зависимости от типа партнёра

## Стек
- HTML + Tailwind CSS (CDN) + Alpine.js (CDN) + Chart.js (CDN)
- Данные: статические JSON-файлы (имитация API)
- Деплой: GitHub Pages
- Только десктоп

## Архитектура
Многостраничный HTML. Каждый модуль — отдельный файл. Общий sidebar через JS-компонент (components/shell.js).

### Структура файлов
```
/Код/
├── index.html          — Дашборд (главная)
├── orders.html         — Заказы
├── claims.html         — Рекламации
├── docs.html           — Документы и договора
├── products.html       — Каталог продукции + API
├── rating.html         — Геймификация и рейтинг
├── components/
│   └── shell.js        — Общий sidebar + header
├── data/
│   ├── orders.json     — Мок-данные заказов
│   ├── claims.json     — Мок-данные рекламаций
│   ├── docs.json       — Мок-данные документов
│   ├── products.json   — Каталог продукции
│   └── dealers.json    — Дилеры, баллы, рейтинг
└── .github/workflows/
    └── pages.yml       — Деплой
```

### Shell-компонент (sidebar)
- Логотип ТРЕК
- Имя дилера + тип (дистрибьютор/СТО)
- Уровень геймификации (иконка + название)
- Навигация: Дашборд, Заказы, Рекламации, Документы, Каталог, Рейтинг
- Счётчики уведомлений на пунктах меню
- Колокольчик уведомлений в header

## Модуль 1: Дашборд (index.html)

### KPI-карточки (4 штуки)
- Заказов за месяц (количество + % к прошлому месяцу)
- Сумма закупок (₽ + % к прошлому месяцу)
- Активных рекламаций (количество + просроченные)
- Геймификация (уровень + баллы + позиция в рейтинге)

### Графики
- Динамика закупок за 6 месяцев (линейный/столбчатый Chart.js)
- Топ-10 товаров по закупкам (горизонтальный bar)

### Рекомендации по закупке
- Блок "Рекомендуем заказать" — товары на основе истории закупок
- Карточки: название, артикул, средний расход, "Осталось ~на N дней/нед."

### Быстрые действия
- Кнопка "Создать заказ"
- Кнопка "Подать рекламацию"
- Документы на подпись (со счётчиком)

### Лента уведомлений
- Хронологический список: отгрузки, новые прайсы, достижения геймификации
- Иконка + текст + время

## Модуль 2: Заказы (orders.html)

### Фильтры
- По статусу, дате, сумме, артикулу

### Таблица заказов
- Колонки: номер, дата, состав (кратко), сумма, статус, трекинг

### Статусная модель заказа
Новый → Подтверждён → В сборке → Отгружен → Доставлен

### Карточка заказа (раскрывается по клику)
- Детали: дата, номер, менеджер
- Состав позиций: артикул, название, количество, цена, сумма
- Документы: счёт, накладная (скачивание)
- Таймлайн статусов

### Аналитика
- Графики продаж по месяцам
- Сравнение с прошлым периодом
- Топ товаров по закупкам

### Прогнозирование
- Рекомендации "на основе ваших закупок рекомендуем заказать через N дней"

## Модуль 3: Рекламации (claims.html)

### Список рекламаций
- Фильтры по статусу, дате, товару
- Таблица: номер, дата, товар, тип проблемы, статус

### Форма создания рекламации
- Номер заказа (выбор из списка)
- Товар (артикул + название)
- Тип проблемы: брак, несоответствие, повреждение при доставке, другое
- Описание проблемы (textarea)
- Фото/файлы (drag & drop зона)
- Количество, номер партии

### Статусная модель рекламации
Черновик → Подана → На рассмотрении → Запрос доп. информации → Одобрена / Отклонена → Возврат/Замена → Закрыта

### Карточка рекламации
- Вся история изменений статуса (таймлайн)
- Переписка (комментарии)
- Прикреплённые файлы
- Текущий статус

## Модуль 6: Геймификация и рейтинг (rating.html)

### Уровни дилеров
- Бронза → Серебро → Золото → Платина → Бриллиант
- Пороги по накопленным баллам

### Начисление баллов
- Заказ оформлен: +10 баллов
- Оплата вовремя: +20 баллов
- 0 рекламаций за месяц: +50 баллов
- Прошёл обучение: +30 баллов
- Объём закупок > 500 тыс. ₽/мес: +100 баллов

### Ачивки-медали
- "Первый заказ"
- "100 заказов"
- "Безупречный квартал" (0 рекламаций за 3 мес.)
- "Миллионер" (1М ₽ закупок)
- "Скорость" (оплата менее 24ч)

### Таблица лидеров
- Топ дилеров по баллам
- Позиция текущего дилера подсвечена
- Анонимизация опциональна

### Прогресс-бар
- До следующего уровня
- Сколько баллов осталось

### Бонусы за уровень (описание)
- Доп. скидка
- Приоритетная отгрузка
- Персональный менеджер
